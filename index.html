C# While Loop Fundamentos

El bucle while repite un bloque de código mientras la condición sea true.

Índice

  1. Descripción
  2. Sintaxis
  3. Ejemplo
  4. Explicación paso a paso

Consejo: pasa el ratón por cada paso para resaltar las líneas del código.

Descripción

Usa while cuando no sabes de antemano cuántas veces iterarás y detienes el bucle al dejar de cumplirse una condición.

Sintaxis

while (condicion) {
    // código a ejecutar
}

Ejemplo

int i = 0;
while (i < 5)
{
    Console.WriteLine(i);
    i++;
}

Explicación paso a paso

  • 1 Inicialización
    Se crea i y se inicia en 0.
  • 2 Condición
    Mientras i < 5, el cuerpo del bucle se ejecuta.
  • 3 Acción
    Imprime el valor actual de i.
  • 4 Actualización
    Incrementa i para progresar hacia la salida.
  • 5 Repetición
    Cuando i llega a 5, la condición es falsa y el bucle termina.

Notas y buenas prácticas

  • Evita bucles infinitos: asegúrate de que la condición cambie dentro del bucle.
  • Si conoces el número de iteraciones, considera for.
  • Para depurar, imprime la variable de control y observa cómo evoluciona.
